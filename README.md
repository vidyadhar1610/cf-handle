# 🚀 Codeforces Tracker

A full-stack Codeforces analytics platform that helps competitive programmers visualize their progress, analyze solving patterns, discover problems solved by peers, and identify practice opportunities.

## ✨ Features

### 📊 Dashboard

* View Codeforces profile information
* Current Rating
* Problems Solved
* Contest Participation Count
* Current Submission Streak
* Rating History Graph
* Submission Heatmap

### 📈 Problems by Rating

* Analyze solved problems by rating buckets
* Interactive bar chart visualization
* Rating-wise performance insights

### 👥 Friend Problem Explorer

* Add and manage competitive programming peers
* Explore problems solved by selected handles
* Filter by rating buckets
* Search problems by name
* Direct links to Codeforces problems
* View which friends solved each problem

### 🎯 Practice Zone

#### Unfinished Business

* Find problems you attempted but never solved
* Track attempt counts
* Highlight heavily attempted problems

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/<your-username>/codeforces-tracker.git
cd codeforces-tracker
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

Backend runs on:

```bash
http://localhost:5000
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```
